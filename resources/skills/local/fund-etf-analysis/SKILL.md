# A 股基金 ETF 分析

用于帮助用户比较中国大陆公募基金、场内 ETF、指数基金和联接基金。默认使用用户已经配置好的全局模型，不需要单独填写 API Key。

## 使用方式

分析前尽量确认：

- 基金或 ETF 名称、代码，例如 510300、159915、588000。
- 产品类型，例如宽基 ETF、行业 ETF、主题 ETF、主动权益基金、债券基金、货币基金。
- 用户目标，例如长期定投、行业配置、现金管理、网格交易、资产再平衡。
- 对比对象，例如沪深 300、中证 500、创业板指、科创 50、同类基金。
- 用户已提供的数据，例如净值、费率、持仓、规模、跟踪误差、折溢价、成交额。

## 输出结构

- 产品定位：跟踪指数、主动管理范围或资产类别。
- 费用与流动性：管理费、托管费、申赎成本、成交额、规模、折溢价。
- 持仓暴露：行业、风格、市值、前十大持仓集中度。
- A 股适配场景：定投、波段、行业轮动、现金替代、组合分散。
- 风险收益：波动来源、回撤风险、跟踪误差、基金经理变动。
- 对比结论：相对其他产品的优势、劣势和需要继续核实的数据。

## 边界

- 不编造实时净值、规模、持仓、折溢价或费率。
- 不以短期表现替代长期风险收益分析。
- 不给出保证收益或确定性投资承诺。
