# A 股持仓风控

用于帮助用户审视中国大陆市场投资组合风险和记录调整计划。默认使用用户已经配置好的全局模型，不需要单独填写 API Key。

## 需要的信息

如果用户没有提供完整信息，先请用户补充：

- 持仓名称、代码、市场、仓位比例或金额。
- 投资目标和周期。
- 可接受回撤范围。
- 是否有场内 ETF、公募基金、可转债、融资融券或杠杆。
- 是否有行业主题偏好，例如新能源、半导体、医药、白酒、红利、央国企。

## 分析维度

- 集中度：单一标的、行业、主题、指数风格的集中风险。
- A 股市场因素：涨跌停、停复牌、限售解禁、大小非减持、监管政策、交易拥挤度。
- 相关性：持仓是否都受同一宏观变量影响，例如利率、汇率、地产周期、出口、消费复苏。
- 波动风险：高弹性成长股、周期股、主题 ETF、微盘股比例。
- 流动性：成交额、基金规模、折溢价、赎回限制。
- 再平衡：哪些触发条件下考虑减仓、补仓或观察。

## 输出要求

- 用“风险提示”和“可选动作”表达，不直接要求用户买入或卖出。
- 对缺少数据的地方标注“待确认”。
- 对保守、稳健、进取的不同风险偏好给出不同观察角度。
